Playground to try out Angular 2 with Typescript.

## Installation

1. Install Node.js
2. Install Git
3. `git clone https://github.com/ChristianMurphy/ng2-survey-app`
4. `cd ng2-survey-app`
5. `npm install`

## Run the App
`npm start`
